.acr "!commands" %user% typed `!commands`. This is a list of available commands, more detailed descriptions of the commands can be found in #about.

**Role Commands**

`.iam Server #` Replace hash with your server, 1 to 18 for international, or korea for korea
`.iam Stage ####` Replace hash with stage role. You can find all self-assignable roles in pinned post
`.iamnot Stage #000`
`.iamnot Server #`
